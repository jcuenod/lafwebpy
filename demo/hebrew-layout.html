1. Regular ol' Hebrew text in a div

פָּדָ֣ה ֯נפשׁי מֵעֲבֹ֣ר בַּשָּׁ֑חַת ֯ו֯חיתי בָּאֹ֥ור תִּרְאֶֽה׃

2. But this is rtl text so let's actually style it as such

פָּדָ֣ה ֯נפשׁי מֵעֲבֹ֣ר בַּשָּׁ֑חַת ֯ו֯חיתי בָּאֹ֥ור תִּרְאֶֽה׃

3. But we need to click/hover on each bit of the word so we wrap all the bits in spans (now try hovering) but note the wrapping around ֯ו֯חיתי - at times the diacritics are separated from the words they go with. Note that there is no masora circle associated with בַּשָּׁ֑חַת.

פָּדָ֣ה ֯נפשׁי מֵעֲבֹ֣ר בַּשָּׁ֑חַת ֯ו֯חיתי בָּאֹ֥ור תִּרְאֶֽה׃

4. So we wrap whole words together (and put their trailers into the spans so that maqef units don't break things etc.) and we set the spans to display as inline-blocks so that they don't break.

פָּדָ֣ה ֯נפשׁי מֵעֲבֹ֣ר בַּשָּׁ֑חַת ֯ו֯חיתי בָּאֹ֥ור תִּרְאֶֽה׃

5. Initially I had thought that the masora circles were supposed to appear on the left. This can be accomplished by making "word bit"s inline-blocks as well and changing their text direction ltr. But I think the solution above is preferable.

פָּדָ֣ה ֯נפשׁי מֵעֲבֹ֣ר בַּשָּׁ֑חַת ֯ו֯חיתי בָּאֹ֥ור תִּרְאֶֽה׃
